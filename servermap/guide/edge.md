# Edge
## Description

Edges are lines connecting nodes. `source` and `target` refer to the node's id.


## Props
| Props | Type  | Required  | Description  |
| --- | --- | --- | --- |
| id | string  | ✔️ | Id of the edge |
| source | string | ✔️ | Source of the edge |
| target | string | ✔️ | Target of the edge |
